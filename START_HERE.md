# 🚀 Quick Start Guide

## How to Run the Web App

### **Option 1: Run with Python (Easiest)**
Open a terminal in this folder and run:
```bash
python run_app.py
```

### **Option 2: Run with Batch File (Windows)**
Double-click:
```
run_app.bat
```

### **Option 3: Run with PowerShell (Windows)**
Right-click and run:
```powershell
.\run_app.ps1
```

---

## What Happens Next

✅ Streamlit will install automatically (if not already installed)  
✅ A web browser will open at `http://localhost:8501`  
✅ You can immediately upload your CSV or Excel file  

---

## Using the Web App

### **Step 1: Upload Your File**
- In the left sidebar, select **"Upload CSV"** or **"Upload Excel"**
- Click to browse and select your file from your PC
- The file will be analyzed automatically

### **Step 2: Explore Your Data**
Click on the tabs to see:
- **📈 Overview** - Dataset summary and metrics
- **🔍 Analysis** - Customer statistics and insights
- **📊 Visualizations** - Interactive charts and graphs
- **🤖 Models** - Machine learning model performance

### **Step 3: Compare Different Files**
Simply upload a new file anytime - the app will refresh with new analysis

---

## Requirements

✓ Python 3.7+  
✓ Internet connection (for first-time package installation)

## To Stop the App

Press **Ctrl+C** in the terminal window
